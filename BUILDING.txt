########################################################################
## Instructions to build PMC on your machine
########################################################################

------------------------------------------------------------------------
-- Main requirements
------------------------------------------------------------------------

OS: Windows, linux, most UNIX* probably

CMake - build system configuration environment

C++ compiler - GCC and MSVC have been tested

Boost headers - PMC only uses boost headers

Boost unit_test_framework - optional for unit testing

------------------------------------------------------------------------
-- Python support requirements
------------------------------------------------------------------------

Python Libs - required for python support

SWIG - required for python support

Numpy - optional for fixed width support

------------------------------------------------------------------------
-- Get the source
------------------------------------------------------------------------
git clone https://github.com/guruofquality/PMC.git

------------------------------------------------------------------------
-- Build and install - UNIX Makefiles
------------------------------------------------------------------------
mkdir <build dir>
cd <build dir>
cmake <source dir>
make
make test
sudo make install

------------------------------------------------------------------------
-- Build and install - MSVC
------------------------------------------------------------------------
Configure dependencies with CMake GUI
Open the generated soltions file:
ALL_BUILD
RUN_TESTS
INSTALL
